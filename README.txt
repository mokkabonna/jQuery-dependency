A jQuery plugin for creating dependencies between checkboxes/radiobuttons

Usage:


...$('.myinputchildren').dependsOn('#parent');

<input type="checkbox" id="parent" />
<input type="checkbox" class="myinputchildren" />
<input type="checkbox" class="myinputchildren" />
...

Read more and see examples at http://martinhansen.no/2011/03/29/jquery-dependance-plugin/
